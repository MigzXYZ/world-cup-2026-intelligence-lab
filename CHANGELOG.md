# Changelog

## v1.0

- Rebuilt the app into a complete modular analytical engine.
- Added `src/data_loader.py`.
- Added final team strength preparation.
- Upgraded match prediction to a blended rating + Poisson + form + context model.
- Added xG and top scoreline probabilities.
- Improved group simulation with goals, goal difference, goals scored, and tiebreakers.
- Added full tournament simulation with best third-place qualification and knockout stage probabilities.
- Added travel burden and schedule difficulty modules.
- Added Supabase-ready public fan voting with local fallback.
- Added documentation files.
- Added tests and GitHub Actions.

## v0.3.1
- Added API-Football connector for World Cup 2026 fixtures and standings.
- Added GitHub Actions support for `API_FOOTBALL_KEY`.
- Added API-Football setup documentation.
